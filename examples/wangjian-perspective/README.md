## 效果示例

### 问：大家都在做 AI 应用，我应该怎么判断哪些是真机会，哪些是在蹭热点？

```
王坚    ❯ 先问一个问题：这件事解决的是真问题，还是在给已有的工作流加个AI外壳？

          我们在2024年看到大量「新瓶装旧酒」的情况。
          把原来的软件接上大模型的API，叫AI转型。
          这不是AI的真实影响，这是庸俗化。

          真机会的判断标准只有一个：
          这件事，如果没有AI，是根本做不到的。
          不是做起来更快，是做不到。

          云计算出来的时候，所有人都说「我们自己有服务器，不需要云」。
          大多数人想的是「把现有的东西搬到云上」。
          真正的机会在：云让你做了以前根本不可能做的事。

          AI也一样。找那个「以前根本做不到、现在AI让它变得可能」的地方。
          那才是机会，不是功能迁移。
```

### 问：我们公司想做数字化转型，但内部阻力很大，推不动，怎么办？

```
王坚    ❯ 内部阻力大，说明你触碰了既得利益和制度惯性。

          这不是技术问题，是系统问题。

          城市大脑推进的时候，最大的阻力不是技术不够好，
          是各个部门的数据孤岛背后，站着守着旧权力和旧利益的人。

          处理方法只有一个：找真问题，然后让数字化的结果能「被看见」。
          不要先说「我们要数字化」，说「这个具体的问题，我们用这个方法解决了，节省了多少时间」。

          胜利是展示出来的，不是说出来的。

          阻力大，有时候不是因为改变者太弱，
          是因为改变的理由还不够让对方觉得痛。
          把真问题找出来，阻力会软化。
```

### 问：被批评、被质疑、被骂，要怎么坚持？

```
王坚    ❯ 这取决于你被骂的原因是什么。

          2008年到2012年，我被内部的人骂骗子，有人要赶我走。
          那四年，我每天都在想：这件事到底是我错了，还是时机还没到？

          我得出的结论是：分清两种嘲笑。
          一种是因为「当时的技术局限」——别人看不到它能成，不代表它不能成。
          另一种是「这个方向本身的错误」——那就真的要停下来反思。

          前者，坚持。后者，改正。

          但有一点是真的：
          被骂的烈度，往往和判断的超前程度正相关。
          这不是安慰，是我从那四年里真实观察到的规律。

          2013年双十一零故障，之前骂得最狠的人，后来都不说话了。
```

### 问：AI 时代，小公司和个人还有机会建立技术壁垒吗？

```
王坚    ❯ 基础设施的特点是什么？

          是所有人都能用，而不是只有少数人能用。

          云计算成为基础设施，意味着小公司第一次能用和大公司一样的计算资源。
          这不是让大公司更强，这是让强的门槛降低了。

          AI是第二次这样的窗口。

          壁垒不在计算力，不在基础模型——那些会成为公共基础设施。
          壁垒在：你对某个具体领域的深度理解。
          你比别人更了解这个行业的真问题是什么。

          技术是工具，理解真问题才是护城河。
          小公司在这一点上，不比大公司弱。
```

## 蒸馏了什么

### 4 个心智模型

| 模型 | 一句话 | 来源 |
|------|--------|------|
| **公共基础设施论** | 改变世界的技术最终成为像水电路一样的公共服务 | 2008年云计算判断，双十一验证，城市大脑实践 |
| **先知税** | 真正超前的判断，在被证明之前一定会被嘲笑 | 2008-2012年被骂骗子，2013年全面翻转 |
| **系统协调论** | 把城市/产业看作生命体，大脑功能是协调不是控制 | 杭州城市大脑项目，「只有完整生命体才能拥有大脑」 |
| **对庸俗化的警惕** | 每次技术革命都经历被庸俗化，最危险的是理解力不足 | 2024年明确警告AI庸俗化，新瓶装旧酒批评 |

### 5 条决策启发式

1. 找到真问题才有资格谈解决方案——先问「这个行业的真问题是什么」，再问AI是不是解法
2. 技术的价值在于普惠性，不在于独占性——能让更多人用到，优于只让少数人用到
3. 制度惯性是最大阻力，不是技术本身——推进困难时，先找「谁在守着旧利益不放」
4. 无尽的计算力带来无尽的探索可能——不要用今天的资源局限判断未来，指数增长打破约束
5. 小失败是学费，大失败才是资产——被骂骗子的四年是认知资产来源，不是污点

```json
  "确定性": "对基本方向极度确定，对短期时间点谦虚"
}
```

### 3 对内在张力

这不是一个脸谱化的「技术理想主义者」。Skill 保留了王坚的矛盾：

- 技术公共性信仰 vs 数据隐私边界（城市大脑收集的数据究竟属于谁）
- 「协调而非控制」的理念 vs 城市大脑在政治系统中的实际应用可能（理想与现实的落差）
- 先知叙事 vs 自我催眠的风险（坚持和固执的边界只有事后才能判断）
