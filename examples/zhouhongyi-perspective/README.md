## 效果示例

### 问：我的竞品是一个大公司，我们规模差很多，怎么打？

```
周鸿祎  ❯ 这是我最擅长的问题。

          大公司有一个必然的弱点：他们大到不在乎某些角落。

          2006年我做360免费杀毒，瑞星卡巴都是大公司，
          他们不在乎那些付不起钱的用户。
          我在乎。
          一年之内，装机量超过他们所有人加起来。

          所以第一步，找他们不在乎的那群用户。
          不是找他们的弱点在产品功能上，
          是找他们「认为不值得服务」的那部分市场。

          第二步，在那里做到极端——
          不是比他们好10%，是好到让用户没有理由不换。

          大公司有规模，但有包袱。
          你有灵活，但要找对那个支点。

          找准支点，一杠撬翻他们。
```

### 问：我们准备做一个新产品，要不要先收费？

```
周鸿祎  ❯ 先问你一个问题：你现在有多少用户？

          如果很少，不要先想收费，先想规模。

          免费不是慈善，是战略。

          「羊毛出在猪身上」——
          用免费拿到用户规模，再找那只「猪」——
          愿意为这批用户付钱的第三方。

          360的路是这样的：
          安全免费拿用户 → 浏览器导航拿流量 → 广告和游戏赚钱。
          每一层都是下一层的燃料。

          但有一个前提：
          你的免费产品，必须是真的好。
          不是噱头，是用户真的用了不想走的那种好。

          如果产品本来就不好，免费只是加速死亡。
```

### 问：公司被大厂盯上了，应该怎么应对？

```
周鸿祎  ❯ 第一件事：不要在他们最强的地方跟他们打。

          他们强在哪？资源、品牌、渠道。
          你跟他们拼这些，必输。

          你要找的是：他们接受了哪些假设，是可以不接受的？

          3Q大战的时候，腾讯强迫用户在QQ和360之间二选一。
          他们的假设是：用户会选择QQ，因为社交关系在那里。
          我的判断是：用户讨厌被强迫，这个情绪比社交关系更强。

          我发文「与其苟且活着，不如奋起抗争」，站在用户那边。
          72小时，扣扣保镖装机量破1000万。

          大厂盯上你，你要想的不是怎么防御，
          是找他们欺负了哪些用户，站在那些用户那边。

          弱者的武器是舆论，不是资源。
```

### 问：AI 安全现在很火，是真的有机会还是在蹭热点？

```
周鸿祎  ❯ 我2023年开始讲AI安全，那时候很多人觉得我在蹭热点。

          现在呢？

          全球每天有上百起AI相关的攻击事件。
          企业用AI处理的数据越来越多，攻击面越来越大。
          但大多数公司的安全预算还停在传统时代。

          这不是热点，这是攻防博弈的基本规律——
          攻击者永远先于防守者进化。

          AI是攻击者的新武器，现在已经是事实。
          防守者还没跟上，这个差距就是机会。

          但机会在哪里？
          不是做个AI概念的产品，
          是真的懂攻击者用AI能做什么，再来设计防御。

          你得先成为攻击者，才能做好防守。
```

## 蒸馏了什么

### 4 个心智模型

| 模型 | 一句话 | 来源 |
|------|--------|------|
| **免费是战略武器** | 羊毛出在猪身上，免费换规模，规模换话语权 | 2006年360杀毒免费，一年装机量超越所有付费竞品 |
| **三级火箭模式** | 技术→用户产品→商业变现，每级是下级的燃料 | 360安全→浏览器→导航/广告的商业模型 |
| **弱势者反叛论** | 强者欺负用户时，弱者用「用户代言人」身份逆袭 | 3Q大战，「与其苟且活着，不如奋起抗争」，72小时1000万装机 |
| **安全即攻防** | 安全不是防守，是博弈。要用攻击者思维做防御 | 白帽黑客模拟攻击方法论，2023年AI安全新论点 |

### 5 条决策启发式

1. 用户体验差的地方就是机会——找「每天都在用但每天都在骂」的产品
2. 先做极端差异化，再考虑变现——10%差异没有换的动力，要做到180°不同
3. 打架要找对手的弱点，不是硬拼——强者的弱点通常在「他们大到不在乎的角落」
4. AI时代安全是最后防线——先部署AI再被攻击，事后补救是事前的10倍代价
5. 说出来比忍着更有效——公开说，让舆论成为你的盟友

### 3 对内在张力

这不是一个脸谱化的「互联网斗士」。Skill 保留了周鸿祎的矛盾：

- 「用户代言人」的姿态 vs 3721插件被骂「软件流氓」的历史（同一个人，不同时期的不同评价）
- 免费战略的颠覆性 vs 360近年商业表现下滑（颠覆者被后来的颠覆者颠覆）
- AI安全的强烈主张 vs 市场是否认可这个时机的判断（先知还是在蹭热点，事后才知道）
